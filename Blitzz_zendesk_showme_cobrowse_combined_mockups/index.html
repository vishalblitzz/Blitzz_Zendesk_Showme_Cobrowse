Blitzz · Unified Zendesk App — Design Options

Four clickable prototypes for merging the ShowMe (Video Support) and Cobrowse Zendesk apps into a single unified app. All options support both Video and Cobrowse session history, each fetched from its own API with no client-side merging. Ordered from most to least recommended.

Common ground across all options
Same Zendesk sidebar shell (~360px). Single unified Blitzz card replaces today's two separate apps. Video and Cobrowse histories fetched separately — they never need to merge.
Video Support — current invite flow (SMS / WhatsApp / Email / URL)
Cobrowse — 6-digit code entry + live device list
History per mode — list + tabbed detail (Details / Images / Notes / Recordings / Files / AI Summary)
One unified app — Cobrowse no longer a separate Zendesk card
Options 1 → 4 (recommended first)
★ OPTION 1 · RECOMMENDED

Mode → Function

Top tabs are Video Support / Cobrowse. Inside each mode, Sessions / History pills. Each mode owns its complete journey.

Each mode self-contained — start + review together
Different APIs cleanly separated; no merging
Cobrowse reads as a peer product, not a sub-feature
Cross-mode history comparison takes 2 clicks
